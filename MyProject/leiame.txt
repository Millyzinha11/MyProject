# GitHub

O GitHub é um programa voltada para programação e desenvolvimento de códigos colaborativo.
Dentro do GitHub,podemos contruir repositorios baseados nos fluxos de tabalho e torná-los públicos ou privados.
É uma plataforma que possibilita a extensão de trabalho, podendo ser feito trabalhos em conjuntos de forma separada podendo se unificar depois.
Também é uma biblioteca moderna de JavaScript.
Aréa com modelos de extensões podendo ser utilizadas para a criação de códigos.
